Place PNG icons in this directory with exactly these filenames:

- `favorites.png`: favorites category icon
- `status.png`: status/system overview icon
- `scene.png`: scene/camera icon
- `keys.png`: keyboard/controls icon
- `joints.png`: joints/body structure icon
- `actuators.png`: actuators/power/motor icon
- `favorite_on.png`: starred state
- `favorite_off.png`: unstarred state

Notes:

- Use PNG only.
- Square icons work best, recommended size: `32x32` or `64x64`.
- Prefer transparent background.
- These icons are used by the MuJoCo GUI activity bar and favorite toggle.
